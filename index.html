Habla Conmigo!

Es una app gratis que sirve como herramienta de comunicación para personas en todo el mundo.

Caracteristicas principales

Texto a Voz

Voz a Texto

Lenjuage de señas

Pictogramas

  • Frases prearmadas para facilitar la comunicación.
  • Se pueden seleccionar emoticones (emojis), que también la aplicación los reconoce y los dirá.
  • Una guía informativa sobre el mutismo selectivo.
  • Una sección para aprender o tener a mano el lenguaje de señas.
  • Pictogramas para facilitar la comunicación. En esta sección cuando se toca la carta emite la palabra escrita en la misma.
